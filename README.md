# Social Orbit 🪐

Social Orbit is a modern, location-based social networking platform built with React. It aims to connect individuals through localized events, shared interests, and a gamified trust system. Designed with a sleek, minimalist glassmorphism UI, Social Orbit provides a premium and intuitive user experience.

## ✨ Features

- **Interactive Social Radar:** A real-time heatmap showing active social zones and hotspots around your city.
- **Dynamic Event Feed:** Discover and filter local events such as acoustic concerts, theater plays, coffee tastings, and outdoor activities.
- **Skill-Tree & Badges:** A built-in RPG-style progression system. Users unlock achievements, grow their "Skill-Tree" (Music, Culture, etc.), and earn distinct badges.
- **Trust Score System:** An integrated trust and reputation mechanism ensuring a safe and reliable community environment.
- **Glassmorphism UI:** A highly polished, modern user interface built from scratch using CSS variables and Framer Motion for smooth animations.

## 🚀 Technologies Used

- **Frontend Framework:** React 18 (Vite)
- **Styling:** Pure CSS (Custom variables, Flexbox/Grid, Glassmorphism aesthetics)
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 🛠️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/nazarcabir/Social-Orbit.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Social-Orbit
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/App.jsx`: The core application file managing routing, state, and main UI components (Home Feed, Radar, Profile).
- `src/index.css`: The global stylesheet defining the theme, color palette (Ice Blue & Baby Blue), and glassmorphism utilities.
- `src/main.jsx`: The entry point for the React application.
